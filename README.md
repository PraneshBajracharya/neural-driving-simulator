# Neural Driving Simulator

A browser-based self-driving car simulation using JavaScript, ray-cast sensors, collision detection, and a simple neural network controller.

This project is based on Radu Mariescu-Istodor's Self-Driving Car JavaScript course. I used the original tutorial project as a foundation and extended it with my own features, refactoring, and experimentation.

## My Additions

- Added configurable training controls for mutation rate, car count, and traffic density

- Added simulation metrics including distance traveled, crashes, and best fitness score

- Refactored the project structure for readability

- Added model save/load/reset behavior

- Improved documentation and setup instructions
